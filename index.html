AdGuard Home

DNS blocklists

Filters → DNS blocklists → Add blocklist → Add a custom list → paste the adblock.txt raw URL.

NextDNS

Denylist / custom

Security → Add a blocklist → paste the domains.txt raw URL as a custom list.

Pi-hole

Adlists

Group Management → Adlists → paste the hosts.txt or adblock.txt raw URL → then pihole -g.

uBlock Origin

Filter lists

Dashboard → Filter lists → Import → paste the adblock.txt raw URL → Apply changes.
